PLANDIRNG-SASS

Esta web desarrollada en Angular, pretende mostrar el template que se ha creado con el proyecto PLANDIRNG SASS.
Este proyecto es la base de los estilos del nuevo plan director, tanto a nivel de componentes, como a nivel de layout, paginas de acceso, etc..
Esta desarrollado en SASS, para utilizar las ventajas de, estructura del proyecto con partials, utilizacion de variables, funciones y mixins.
El resultado o compilacion del proyecto genera el archivo PLANDIRNG.css, que es el que importaremos a nuestro proyecto Angular.
