PLANDIRNG

Esta web desarrollada en Angular, pretende mostrar el template que se ha creado con el proyecto PLANDIRNG SASS.
Este proyecto es la base de los estilos del nuevo plan director, tanto a nivel de componentes, como a nivel de layout, paginas de acceso, etc..
Está desarrollado en SASS, para utilizar las ventajas de, estructura del proyecto con partials, utilización de variables, funciones y mixins.
El resultado o compilacion del proyecto, genera el archivo PLANDIRNG.css, que es el que importaremos a nuestro proyecto Angular, y el que hemos importado en esta aplicación, para ver el aspecto final de todos los componentes.

PLANDIRNG utiliza el core de primeng y el primeicons, ya que mucho de los componentes que se han ido creando, internamente hacen referencia a los iconos de prime. Por lo tanto debemos tener instaladas estos dos paquetes. 


                  npm install primeng --save
                  npm install primeicons --save
PLANDIRNG en su base, incluye una serie de librerias, que no necesitamos importar y que forman parte de su core.
La ventaja fundamental es que tendremos un único archivo .css (y una única llamada al servidor con todo lo que necesitemos).
